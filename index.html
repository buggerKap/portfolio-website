Sphere Mesh Generation

This project is prototype for upcoming project, in which would depicted an simulation of the solar system. For this project instead of storing a 3D model of mesh but instead it uses trigonogemetry principles to generateda spherical mesh.

By generating the mesh the program is given more freedom over the actual object, allowing for the same application to render a high or low poly mesh, without the need to read in from memory.

Lab Rat Chronicles

Lab Rat is an 3rd person platformer, in which the player plays as rat going through a series of 3 tests, the project was meant to explore the ethical concerns of animal testing via the environment and gameplay mechanics.

The player can find collectables objects and customize their rat with unique hats inside their hub.
